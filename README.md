# SQL Data Analysis Practice

This repository contains SQL solutions to advanced data analysis problems.

The goal is to practice real-world analytical SQL skills used in data analyst and data scientist interviews.

## Topics covered

- Aggregations
- Window Functions
- Ranking
- Recommendation systems
- Time series analysis
- Multi-table joins
- CTEs

## SQL concepts used

- JOIN
- GROUP BY
- HAVING
- CTE
- LAG / LEAD
- DENSE_RANK
- Window Functions
